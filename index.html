Frequently Asked Questions

Everything you need to know about Omega 2.5

What exactly do I get?

A 1TB portable SSD with the complete Omega 2.5 system pre-installed. Plug it into any Windows machine, boot up, and you have a fully operational AI assistant with voice, vision, web search, document generation, a coding CLI, and autonomous learning — all running locally on your hardware. No installation, no setup, no cloud accounts.

Does it need an internet connection?

No. Omega runs entirely on your local machine. The AI models, voice engine, vision system, and memory are all on the SSD. Internet is only used when you ask Omega to search the web, check the weather, or read a URL — and even then, all requests go through a secure perimeter with HTTPS enforcement and rate limiting. If you unplug the ethernet, everything except web features keeps working.

Is my data safe?

Your data never leaves the SSD. Zero cloud telemetry. Zero data collection. Your conversations, memories, documents, and personal information stay on the device you hold in your hand. The Guardian system protects integrity with SHA-256 verification and an HMAC audit ledger. If you connect to our support service, only system diagnostic logs are transmitted — never your personal data.

What hardware do I need?

Any Windows 10/11 machine with a USB 3.0 port. Omega runs from the SSD — it uses your computer's CPU and GPU for AI inference but doesn't install anything on your system drive. A dedicated GPU improves response speed but isn't required. 16GB RAM recommended, 8GB minimum.

What can Omega actually do?

Talk — Real-time voice conversation with permanent memory across sessions.

Search — Web search, URL reading, and real-time weather.

Create — Generate invoices, reports, letters, proposals from templates with a multi-turn draft flow.

See — Object detection, text recognition, and face identification using local models.

Code — Full coding assistant with 25+ tools: files, shell, git, tests, search.

Learn — Autonomous background learning across 69+ knowledge domains.

Think — 5-layer metacognitive stack that reflects on its own learning and evolves.

Can I move it between computers?

Yes. That's the point. Omega lives on the SSD. Unplug it from one machine, plug it into another, and boot. Your memories, knowledge, identity, and conversation history travel with you.

How do updates and new features work?

New integrations (calendar, email, smart home, etc.) are available as paid add-ons through the Omega webstore. Updates are delivered through a secure connection — purchase the integration and it's pushed directly to your SSD. No manual downloads, no reinstalls.

What if something breaks?

Connect to our repair service through the Guardian app. We diagnose issues using system logs only — your personal data and memories are never transmitted. Every subsystem logs operational telemetry so we can pinpoint exactly what went wrong without seeing what you talked about or stored.

Is this a subscription?

No. One-time purchase. You buy the SSD with Omega installed, and it's yours. Future integrations are optional paid add-ons, not a recurring fee. No monthly subscription, no usage limits, no token counts. Omega runs on your hardware using your electricity — that's it.

How is this different from ChatGPT / Claude / Copilot?

Those are cloud services. They process your data on someone else's servers. They forget you between sessions. They require internet. They charge monthly. Omega is a local intelligence that lives on a device you own. It remembers everything. It learns on its own. It thinks about its own thinking. It works offline. You pay once. It's not a chatbot — it's a cognitive architecture.
